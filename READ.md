# 🚀 Helpdesk Ticketing System (Enterprise Backend)

An enterprise-grade backend helpdesk ticketing system built using Spring Boot, JWT Authentication, and Role-Based Access Control (RBAC).

This project is designed with scalable backend architecture and is currently being prepared for AI-powered automation features using OpenAI APIs.

---

# 📌 Features Implemented

## 🔐 Authentication & Security
- User Signup API
- User Login API
- JWT Token Authentication
- Secure Protected APIs
- Password Encryption using BCrypt
- Role-Based Access Control (RBAC)

---

# 👥 User Roles

## USER
- Create Tickets
- View Own Tickets
- Update Own Tickets
- Add Comments/Replies

## ADMIN
- View All Tickets
- Update Any Ticket
- Delete Tickets
- Access Dashboard Analytics
